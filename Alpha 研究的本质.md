## Alpha 研究的本质是什么

找市场的**系统性低效**——别人反复犯的错，你反复赚。

关键词是"反复"。一次的不算，运气的不算，必须是统计上可重复的模式，并且要能**解释为什么会存在**（如果解释不了，大概率是过拟合）。

## 具体建议：按层次推进

### 第一层：先测已知因子，不要自己发明

你有 1666 只股票的日线数据，可以直接验证学术界已知在 A 股有效的因子：

| 因子         | A股有效性      | 逻辑          |
| ---------- | ---------- | ----------- |
| 短期反转（1个月）  | ✅ 强        | 散户追涨杀跌后均值回归 |
| 动量（3-12个月） | ⚠️ 弱/不稳定   | A股动量效应弱于美股  |
| 小市值        | ✅ 强（但近年减弱） | 流动性溢价       |
| 低波动率       | ✅ 稳定       | 行为偏差导致高波被高估 |
| 涨停次数/板块热度  | ✅ 短线有效     | A股特有，散户情绪驱动 |
| 分析师预期上调    | ✅ 中长线有效    | 信息扩散慢       |

**先做这件事：** 对每个因子做 IC 分析（信息系数），看预测能力有多强、有多稳定。

### 第二层：建一个因子测试流水线

你现在的系统缺这个模块：

```
原始数据 → 因子计算 → IC/ICIR 评估 → 因子组合 → 策略信号
```

核心指标只有两个：

- **IC**（信息系数）：因子值和下期收益的相关性，>0.05 就值得关注
- **ICIR**（IC 的信噪比）：IC均值/IC标准差，>0.5 说明因子稳定

这比你现在用的 Sharpe 回测更快、更早能判断因子有没有预测能力。

### 第三层：用组合替代单策略

测出3-5个低相关因子之后，不要选一个最好的，而是**加权组合**。原因很简单：没有一个因子在所有市况下都有效，组合可以平滑。

## 最推荐的学习路径

**读这两个东西，够用很长时间：**

1. **WorldQuant 的 101 Alphas 论文**（免费，搜"101 formulaic alphas"）
   
   - 101个量化因子公式，直接可以在你的数据上实现和测试
   - 这是你最快能上手批量测试的起点

2. **《因子投资：方法与实践》** — 石川、刘洋溢、连祥斌
   
   - 专门写 A 股因子研究的中文书，非常实用
   - 涵盖 IC 分析、因子合成、组合构建全流程
