# Alpha研究三阶段：交易逻辑与工程对应关系

每一步工程动作都必须能回答：它在解决什么交易认知问题？

---

## Phase 1：建立IC基线

### 核心交易问题

> **我现在用的策略信号，到底有没有预测能力？预测的是什么、预测多远？**

突破策略（entry25/exit20）用的是价格动量。但"感觉有效"和"可量化证明有效"之间有鸿沟。IC分析把这个鸿沟变成一个可测量的数字。

### Phase 1 研究的3个基础因子

| 因子 | 交易逻辑含义 | 预期IC方向 | 预期生效周期 |
|---|---|---|---|
| **20日动量**（ret_20d） | 强者恒强——过去涨的继续涨 | 正IC（买强势） | 5-20日 |
| **换手率**（turnover_20d） | 高换手 = 市场分歧大，情绪主导，短期可反转 | 负IC（换手高则短期走弱） | 1-5日 |
| **市净率**（pb_ratio） | 低估值 = 安全边际，长期均值回归 | 负IC（低PB → 未来涨） | 20日以上 |

选这3个的原因：它们代表三种截然不同的交易逻辑（动量/情绪/估值），IC方向和生效周期各不相同，用来验证数据管道和工具链，而不是寻找真正的Alpha。

### 工程对应

| 交易逻辑 | 工程实现 | 输出指标 |
|---|---|---|
| 因子有没有预测力 | alphalens IC分析 | IC均值（目标：绝对值>0.03） |
| 预测力稳不稳定 | ICIR = IC均值/IC标准差 | ICIR（目标：>0.5，月频） |
| 预测力能持续多久 | IC衰减曲线 | 衰减半衰期（目标：>5个交易日） |
| 买哪类的比买哪类的强 | alphalens分组收益分析 | 分组收益单调性 |

## Phase 2：系统化因子挖掘

### 核心交易问题

> **市场上哪些可重复的价格行为模式，可以被我提前发现和利用？**

这是Alpha研究的主体。每个候选因子都对应一个关于市场的假说。因子筛选本质上是在用数据检验你对市场的认知是否正确。

### Phase 2 因子分类与交易逻辑

#### 类别A：价格动量类

| 因子 | 交易逻辑假说 | 缝隙假说格式 |
|---|---|---|
| N日动量（5/10/20/60/120日） | 趋势惯性，机构建仓不是一次性完成的 | 我怀疑：机构分批建仓导致的价格延续性，使得近期强势股在接下来N周仍跑赢大盘 |
| 动量反转（1日反转） | 日内流动性提供者推高了短期反弹 | 我怀疑：A股散户追高行为在当日冲高后必然遇到做市商对冲，导致次日小幅回调 |
| 52周高点效应 | 心理阻力位，突破高点吸引更多买盘 | 我怀疑：价格突破52周新高后，被媒体和散户关注度的放大，会推动价格继续上行 |

#### 类别B：成交量/流动性类

| 因子 | 交易逻辑假说 | 缝隙假说格式 |
|---|---|---|
| 量价背离（涨价缩量） | 涨势缺乏承接，大概率见顶 | 我怀疑：A股的涨价缩量是机构出货的信号，散户不知情继续买入推高，但随后量能枯竭导致回调 |
| 换手率变化（换手加速） | 情绪升温，短期波动放大 | 我怀疑：换手率突然加速意味着市场进入博弈模式，短期价格随机性增大，此时持仓风险大于机会 |
| 大单净流入 | 机构资金流向，有信息优势 | 我怀疑：大单净流入表示机构有内部催化剂判断，会在2-3周内推动价格上行 |

#### 类别C：基本面类

| 因子 | 交易逻辑假说 | 缝隙假说格式 |
|---|---|---|
| 市净率（PB） | 低估值安全边际 | 我怀疑：A股对价值因子的反应有长达数月的滞后，低PB股票在资金轮动时会迎来补涨行情 |
| ROE变化趋势 | 盈利质量改善 | 我怀疑：ROE连续改善2个季度的公司，其财报信息被市场充分消化需要3-6个月 |
| 盈余意外（SUE） | 业绩超预期后的漂移效应 | 我怀疑：A股机构一致预期偏低时，超预期业绩会在公告后20个交易日持续推动价格上行 |

#### 类别D：资金流/情绪类

| 因子 | 交易逻辑假说 | 缝隙假说格式 |
|---|---|---|
| 北向资金净买入 | 外资有长期视角，是A股的信息优势方 | 我怀疑：北向连续净买入某行业，会在2-4周内引发国内机构的跟随建仓行为 |
| 融资余额变化 | 杠杆资金情绪反映，快速增加是见顶信号 | 我怀疑：融资余额快速增加后的2周内，市场脆弱性上升，跌幅会显著放大 |

### Phase 2 筛选流水线对应的交易逻辑

```
候选因子（40-80个）
    ↓ IC筛选
    问题：这个因子有没有统计显著的预测能力？
    标准：|IC均值| > 0.03（月频），ICIR > 0.5
    ↓ 衰减检验
    问题：这个因子的预测力能撑多久，决定持仓周期
    标准：半衰期 > 5个交易日（否则交易成本吃掉Alpha）
    ↓ 时间稳定性检验
    问题：这个因子是因为某段特殊行情才有效，还是持续有效？
    标准：分年度IC均为正，没有某一年独大
    ↓ 相关性去重
    问题：这两个因子说的是不是同一件事（重复信息无用）
    标准：因子相关性 < 0.7，保留ICIR更高的
    ↓
有效因子库（预期：20-40个入库）
```

## Phase 3：信号集成

### 核心交易问题

> **如何把多个有预测力的因子，整合成一个能实际下单的交易信号？**

单因子IC不到0.1（业界顶级水平），无法单独使用。因子组合的目的是用不相关的多个弱信号叠加，形成更稳定的复合预测。

### 信号集成的交易逻辑

**为什么要组合**：
- 动量因子：牛市有效，熊市反转
- 估值因子：熊市有效，牛市无效
- 两者结合：在更多市场状态下有效

**怎么组合**：
```
composite_signal = w1 × rank(动量因子) + w2 × rank(换手率因子) + w3 × rank(估值因子) + ...
```

Rank化的原因：消除不同因子的量纲差异，使权重可以直接比较。

### Phase 3 工程对应的交易逻辑

| 交易问题 | 工程实现 |
|---|---|
| 什么组合权重最好？ | Optuna搜索最优w_i（目标：最大化组合IC） |
| 组合信号是真Alpha还是过拟合？ | WFO验证：训练集找权重，测试集验证IC是否保持 |
| 组合信号和当前Gate系统怎么配合？ | Gate=阻断时信号挂起；Gate=允许时取top N候选标的 |
| 组合的哪个因子真正贡献了收益？ | pyfolio绩效归因：分解各因子对最终收益的贡献 |

### Phase 3 → 候选方案输出格式

```
factor_combo_v1：
  成分因子：动量(w=0.4) + 换手率反转(w=0.3) + 低PB(w=0.3)
  选股逻辑：沪深300成分内，复合评分top 10%
  持仓周期：10-20个交易日
  Gate过滤：DrawdownGate + TrendGate（已有）
  样本外IC：0.041（WFO验证）
  历史回测Sharpe：[VBT结果]
  候选方案状态：待前向模拟
```

这个输出会直接进入现有的ExperimentRun + 候选方案档案。
