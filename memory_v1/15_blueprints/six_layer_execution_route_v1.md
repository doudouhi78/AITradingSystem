# 6 层实现路线蓝图 v1

## 一、定位

本文件是当前工程统一实现路线。

它回答三件事：
- 6 层分别解决什么
- 哪些必须自己掌握
- 哪些必须借成熟开源能力

## 二、6 层路线

### 第 1 层：策略孵化
自己掌握：
- Edge 定义
- 获利来源
- 规则原型
- 微观仓位哲学
- 内生风控哲学

可借：
- TA-Lib / Pandas-TA
- VectorBT 作为逻辑原型沙盒

### 第 2 层：数据处理与特征工程
自己掌握：
- 数据清洗规则
- 特征配方
- 质检标准

借成熟能力：
- AkShare
- Pandas
- Pandera
- Qlib 数据处理能力

### 第 3 层：离线回测
自己掌握：
- 成本口径
- 约束条件
- 红线标准

借成熟能力：
- VectorBT
- QuantStats（后续可接）
- Qlib 回测记录与分析能力

### 第 4 层：压力测试与参数优化
自己掌握：
- 过拟合过滤标准
- 参数高原判定标准
- 样本外衰减红线

借成熟能力：
- Optuna
- Qlib workflow / recorder / report

### 第 5 层：仿真交易
自己掌握：
- 回测与仿真偏差解释
- 仿真通过标准

借成熟能力：
- VN.py 或 QMT

### 第 6 层：实盘上线与风险监控
自己掌握：
- 总账户风控规则
- 熔断规则
- 漂移监控标准

借成熟能力：
- VN.py 或 QMT
- 官方 SDK

## 三、主对象层边界

项目继续自持：
- 研究题目
- 规则表达
- 数据快照
- 验证记录
- 风险与仓位说明
- 执行约束
- 正式复审
- 决策状态
- 方案案卷
